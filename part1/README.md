-   运行环境

    Atlas 200 DK 通过制卡操作完成硬件运行环境的搭建。若已制作过 SD 卡，且已存在的系统软件为 1.75.xx.xx 版本（版本号查询方法可参见[查看开发者板系统软件的版本](atlased_04_0033.html)），则无需重新制卡，可参见[升级 Atlas 200 DK](atlased_04_0029.html)进行升级操作。
